# Changelog

All notable changes to SafeMacCleaner will be documented in this file.

## [3.0.0] - 2025-11-16

### Added - Complete Edition
- **Merged all features** from Professional and Advanced editions
- **Unified menu system** with core and advanced feature sections
- **Complete system clean option** that runs all features sequentially
- **Comprehensive configuration** with all options in one place
- **Enhanced logging** with detailed operation tracking
- **Improved safety confirmations** and user guidance

### Features Included
- Time Machine snapshot management
- System and application cache cleaning
- File organization with age-based processing
- Dynamic application protection system
- Duplicate file finder with MD5 checksums
- Big files finder (100MB+ threshold)
- Privacy protector for browser and system data
- Advanced disk cleaner with deep analysis

### Technical Improvements
- **965 lines of code** with comprehensive functionality
- **33KB script size** with all features included
- **No redundancies** - merged duplicate functions
- **Unified protection system** - single app detection
- **Consistent error handling** throughout all features
- **Comprehensive testing** - all functions validated

## [2.0.0] - 2025-11-16

### Added - Advanced Edition
- **Duplicate Finder** - MD5 checksum-based duplicate detection
- **Big Files Finder** - Locate files larger than configurable threshold
- **Privacy Protector** - Clean browser history, cookies, system traces
- **Advanced Disk Cleaner** - Deep system-wide cache analysis
- **Interactive menu system** for feature selection
- **Comprehensive logging** with timestamped files

### Technical Details
- Scans common directories for duplicates (Desktop, Downloads, Documents, Pictures)
- Configurable file size thresholds for big files and duplicates
- Supports Safari, Chrome, Firefox privacy data cleaning
- Advanced cache categorization and analysis
- Safe deletion with user confirmation prompts

## [1.0.0] - 2025-11-14

### Added - Professional Edition
- **Dynamic Application Protection** - Numbered selection of apps to protect
- **Time Machine Snapshot Management** - Delete all or keep specified number
- **System Cache Cleaning** - User Library/Caches and system caches
- **Application Cache Analysis** - Interactive cache deletion with protection
- **File Organization** - Desktop/Downloads with 30+ day age analysis
- **Developer Cache Cleanup** - npm, yarn, pip, poetry cache cleaning
- **Comprehensive Safety System** - Protected directories and apps

### Core Features
- Scans all installed applications automatically
- User selects protection via numbered interface (1,3,5,12 format)
- Interactive file processing for old files (30+ days)
- Safe deletion with backup options
- Detailed space tracking and reporting
- Dry run mode for safe testing

### Safety & Protection
- Essential directory protection (Documents, Desktop, System folders)
- Dynamic app detection (no hardcoded assumptions)
- Interactive confirmations for all deletions
- Comprehensive logging of all operations
- Backup system for large files before deletion
